# Suggested Commands
- `npm install` – install dependencies (Next 15, React 19, TypeScript, ESLint, Tailwind 4).
- `npm run dev` – start the Next.js App Router dev server (Turbopack) on http://localhost:3000 for live previews.
- `npm run build` – create an optimized production build (useful before deploying to Netlify/Vercel).
- `npm run start` – serve the production build locally.
- `npm run lint` – run ESLint (Next config) to catch TypeScript/React/style issues.
- Common shell utilities on macOS/Darwin: `ls`, `rg`, `npm`, `git`, `sed` are available for navigation and code search.
